## 🚀 Instalando clube-de-membros

Para instalar o clube-de-membros, siga estas etapas:

Linux:
```
$ git clone <este projeto>
$ npm install ou yarn install
```

Windows:
```
$ git clone <este projeto>
$ npm install ou yarn install
```

## ☕ Usando clube-de-membros

Para usar clube-de-membros, siga estas etapas:
